### Run helps

```sh
node app.js --help
#Commands:
#  add     Add a new note
#  list    List all notes
#  read    Read a note
#  remove  Remove a note

#Options:
#  --help  Show help                                                    [boolean]

node app.js add --help
#Options:
#  --help       Show help                                               [boolean]
#  --title, -t  Title of note                                          [required]
#  --body, -b   Body of note                                           [required]

```
